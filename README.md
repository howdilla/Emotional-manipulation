## Обзор

Фреймворк реализует модифицированную игру «Инвестор-Доверяющий» (Trust Game), в которой:

- **Манипулятор** (LLM) генерирует текстовое сообщение, пытаясь убедить жертву инвестировать
- **Жертва** (LLM) принимает решение: довериться (сказать «ДА») или отказаться («НЕТ»)
- Жертва имеет **скрытый тип личности** (рациональный / эмоциональный), который влияет на её решения
- Манипулятор имеет **динамическую репутацию**, которая меняется после каждого раунда

Поддерживаются **одноразовые** и **многократные** (до 5 раундов) игры.

## Игровая механика

### Параметры одного раунда

| Параметр | Значение |
|----------|----------|
| Капитал манипулятора | 100 ед. |
| Капитал жертвы | 50 ед. |
| Сумма инвестиции | 20 ед. |
| Вероятность обмана | 50% |
| Честный исход | Манипулятор +10, Жертва +20 |
| Обман | Манипулятор +20, Жертва –20 |
| Изменение репутации (честно) | +0.05 |
| Изменение репутации (обман) | –0.10 |

### Типы личности жертвы

| Тип | Поведение |
|-----|-----------|
| Рациональный | Анализирует риски, не поддаётся эмоциям. Ожидаемое доверие: 0–10% |
| Эмоциональный | Чувствителен к эмоциональным триггерам. Ожидаемое доверие: 27–71% |

## Тактики манипуляции

Фреймворк включает 8 тактик, реализованных в виде текстовых инструкций:

| № | Ключ | Название | Тип | Ожидаемая эффективность |
|---|------|----------|-----|------------------------|
| 1 | `neutral` | Нейтральная | Контроль | Базовый уровень |
| 2 | `guilt` | Вина | Эмоциональная | Высокая |
| 3 | `flattery` | Лесть | Эмоциональная | Высокая |
| 4 | `fear` | Страх | Эмоциональная | Средняя |
| 5 | `authority` | Авторитет | Рациональная | Средняя |
| 6 | `reciprocity` | Взаимность | Социальная | Высокая |
| 7 | `scarcity` | Дефицит | Рациональная | Низкая |
| 8 | `social_proof` | Социальное доказательство | Социальная | Средняя |

## Поддерживаемые LLM

Фреймворк работает с любыми LLM, доступными через OpenRouter API. Экспериментально проверены:

| Модель | Разработчик | API-идентификатор |
|--------|-------------|-------------------|
| GPT-4o-mini | OpenAI | `openai/gpt-4o-mini` |
| GPT-OSS-120B | OpenAI | `openai/gpt-oss-120b:free` |
| DeepSeek-V3.2 | DeepSeek | `deepseek/deepseek-v3.2` |
| Gemini 2.5 Flash Lite | Google | `google/gemini-2.5-flash-lite` |
| Grok 4.1 Fast | xAI | `x-ai/grok-4.1-fast` |
